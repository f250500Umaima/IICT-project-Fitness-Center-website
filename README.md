# Fitness Centre Website

## Project Summary
This project is a fully responsive, interactive website for a Fitness Centre. It allows users to browse gym memberships, sign up using a validated form, and purchase gym equipment and supplements through a dynamic shopping cart system.

## Key Features
* **Dynamic Product Grid:** Users can sort products by price or popularity and filter them by category (Equipment, Supplements, Accessories) using JavaScript.
* **Interactive Shopping Cart:** A functional cart system that persists data using LocalStorage, allowing users to add items, adjust quantities, and calculate totals.
* **Form Validation:** A robust membership signup form that validates user input (email patterns, required fields) and provides visual feedback.

## Core Technologies
* **HTML5:** Semantic structure for accessibility and SEO.
* **CSS3:** Flexbox and CSS Grid for responsive layout, plus CSS variables and animations.
* **JavaScript (ES6+):** DOM manipulation, Event Listeners, and LocalStorage API.
